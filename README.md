rogue_adventure
=====

一个小游戏，使用 [behavior3editor](https://github.com/behavior3/behavior3editor) 编辑生成行为树

Version
-----
Eralng/OTP 22

Build
-----

    $ rebar3 compile

Run
---

    $ sh run.sh
    1> test:run().
    <0.88.0>
    2> test:live().
